# 🍔 Burguer Q'Komo? — Carta Digital

Web estática para el restaurante de comida rápida **Burguer Q'Komo?**. Desarrollada con HTML, CSS y JavaScript vanilla, sin frameworks ni dependencias externas.

## 🌐 Demo

🔗 [Ver en Vercel](https://qkomo.vercel.app/#) 
---

## 📋 Descripción

Carta digital diseñada para mostrar el menú completo del restaurante de forma clara, moderna y accesible desde cualquier dispositivo. El diseño sigue la identidad visual del local: fondo negro con tipografía bold y paleta naranja-rojiza.

---

## ✨ Características

- Diseño minimalista dark mode con paleta naranja-rojiza
- Navegación sticky con scroll suave a cada sección
- Hero section con información del local (teléfono, horario, delivery)
- Menú completo organizado por categorías:
  - Para Picar
  - Hamburguesas
  - Mac Pollo
  - Kebabs
  - Perritos
  - Gondolitas
  - Burritos
  - Baguettes y Montaditos
  - Sandwiches
  - Menú Infantil
- Totalmente responsive (móvil y escritorio)
- Sin frameworks ni dependencias — HTML, CSS y JS puro

---

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura y contenido |
| CSS3 | Estilos, variables CSS, Grid, Flexbox |
| JavaScript (vanilla) | Renderizado dinámico del menú |
| Google Fonts | Tipografías Bebas Neue e Inter |
| Vercel | Despliegue y hosting |
